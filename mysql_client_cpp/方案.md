# MySQL C++ 客户端方案

## 项目概述
使用 C++ 开发一个 MySQL 数据库客户端，提供直观的界面操作 MySQL 数据库。

## 技术栈
- **编程语言**: C++17
- **MySQL 连接库**: mysql-connector-c++ (官方驱动)
- **GUI 库**: Qt 5/Qt 6 (跨平台支持)
- **构建工具**: CMake
- **C++ 标准**: C++17

## 界面布局设计

### 整体布局：左右分栏
```
+-----------------------------------------------+
|  菜单栏 (File, Edit, View, Query, Help)      |
+---------------+-------------------------------+
|               |                               |
|   左侧面板    |          右侧面板              |
|               |                               |
|  (数据库树)   |   -----------------------     |
|               |   |    查询编辑器    |       |
|  - 数据库列表  |   -----------------------     |
|    ├─ db1     |                               |
|    ├─ db2     |   -----------------------     |
|    └─ db3     |   |    查询结果    |         |
|               |   -----------------------     |
|  - 表列表     |                               |
|    ├─ table1  |   -----------------------     |
|    ├─ table2  |   |    日志信息    |         |
|    └─ table3  |   -----------------------     |
|               |                               |
+---------------+-------------------------------+
|  状态栏 (连接状态、查询耗时等)                  |
+-----------------------------------------------+
```

### 左侧面板功能
- **数据库连接树**
  - 显示所有已配置的数据库连接
  - 支持展开/折叠查看数据库
  - 显示表、视图、存储过程等对象
- **操作功能**
  - 右键菜单：刷新、新建数据库、删除、属性等
  - 双击连接：快速连接数据库
  - 搜索功能：快速定位数据库或表

### 右侧面板：上下分栏

#### 上部：查询编辑器
- **功能特性**
  - 语法高亮（SQL 关键字、字符串、注释）
  - 代码补全（表名、字段名、SQL 函数）
  - 多标签页支持（同时编辑多个查询）
  - SQL 格式化
  - 历史查询记录
- **工具栏**
  - 执行查询 (F5)
  - 执行选中 (F9)
  - 停止查询
  - 格式化 SQL
  - 清空编辑器

#### 中部：查询结果
- **结果表格**
  - 支持大数据量分页显示
  - 支持排序（点击列头）
  - 支持列宽调整
  - 支持导出（CSV、Excel、JSON）
- **信息面板**
  - 显示受影响的行数
  - 显示查询执行时间
  - 显示结果行数

#### 下部：日志/消息区域
- **日志功能**
  - 显示 SQL 执行日志
  - 显示错误信息（红色）
  - 显示警告信息（黄色）
  - 支持清空日志
  - 支持导出日志

## 依赖库安装

### Ubuntu/Debian
```bash
# Qt 6
sudo apt install qt6-base-dev qt6-tools-dev cmake

# MySQL Connector/C++
sudo apt install libmysqlcppconn-dev

# 或者使用 MySQL 官方包
# 下载 https://dev.mysql.com/downloads/connector/cpp/
```

### CentOS/RHEL
```bash
# Qt 6
sudo yum install qt6-qtbase-devel qt6-qttools-devel cmake

# MySQL Connector/C++
sudo yum install mysql-connector-c++-devel
```

### macOS
```bash
brew install qt@6 mysql-connector-c++ cmake
```

## 编译和运行

```bash
cd mysql_client_cpp
mkdir build && cd build
cmake ..
make
./MySQLClient
```

## 配置文件格式

### 连接配置 (connections.json)
```json
{
  "connections": [
    {
      "name": "本地MySQL",
      "host": "localhost",
      "port": 3306,
      "username": "root",
      "password": "password",
      "database": "",
      "savePassword": false
    }
  ]
}
```

### 应用配置 (config.ini)
```ini
[General]
language=zh_CN
theme=dark
font=Monospace,10

[Editor]
fontSize=12
tabSize=4
autoComplete=true

[Database]
connectTimeout=10

```

## 技术要点

### 1. MySQL C++ 连接
```cpp
#include <mysql_driver.h>
#include <mysql_connection.h>
#include <cppconn/statement.h>
#include <cppconn/resultset.h>

sql::mysql::MySQL_Driver *driver;
sql::Connection *con;

driver = sql::mysql::get_mysql_driver_instance();
con = driver->connect("tcp://127.0.0.1:3306", "user", "password");
```

### 2. Qt 信号槽机制
- 使用 Qt 的信号槽机制处理 UI 事件
- 数据库查询使用多线程，避免阻塞 UI

### 3. 异常处理
- 数据库操作使用 try-catch 捕获异常
- 在日志区域显示错误信息

### 4. 内存管理
- 使用 Qt 智能指针管理对象
- 及时释放数据库资源
